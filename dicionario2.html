Dicionário para iniciantes


Aqui estão algumas palavras que programadores e pessoas de áreas correlatas utilizam no seu dia a dia.

API - Interface de Programação de Aplicativos
Explicação: Um conjunto de protocolos e ferramentas que permitem que diferentes aplicativos se comuniquem entre si.
Exemplo: I'm using the Google Maps API to add maps to my application.
Tradução: Estou usando a API do Google Maps para adicionar mapas à minha aplicação.

Boolean - Booleano
Explicação: Um tipo de dado que pode ser verdadeiro (true) ou falso (false).
Exemplo: I'm using a boolean variable to check if the user is logged in.
Tradução: Estou usando uma variável booleana para verificar se o usuário está conectado.

Database - Banco de dados
Explicação: Um sistema de armazenamento de informações organizado em tabelas e campos.
Exemplo: We're using a MySQL database to store user information.
Tradução: Estamos usando um banco de dados MySQL para armazenar informações do usuário.

IDE - Ambiente de Desenvolvimento Integrado
Explicação: Um software que oferece ferramentas para escrever e depurar código.
Exemplo: I'm using Visual Studio Code as my IDE.
Tradução: Estou usando o Visual Studio Code como meu ambiente de desenvolvimento integrado.

Framework - Framework
Explicação: Uma estrutura de código que fornece recursos comuns para um tipo específico de aplicação.
Exemplo: I'm using the Django framework to build a web application.
Tradução: Estou usando o framework Django para construir uma aplicação web.

Object-oriented programming (OOP) - Programação orientada a objetos
Explicação: Um paradigma de programação que se concentra em objetos que contêm dados e métodos.
Exemplo: I prefer object-oriented programming for building complex applications.
Tradução: Prefiro a programação orientada a objetos para construir aplicativos complexos.

Repository - Repositório
Explicação: Um local onde o código-fonte de um projeto é armazenado e gerenciado.
Exemplo: We're storing our code in a Git repository.
Tradução: Estamos armazenando nosso código em um repositório do Git.

Stack trace - Rastreamento de pilha
Explicação: Uma lista de todas as funções que foram chamadas antes de um erro ocorrer.
Exemplo: The stack trace shows where the error occurred in the code.
Tradução: O rastreamento de pilha mostra onde o erro ocorreu no código.

Thread - Thread
Explicação: Um processo separado que pode ser executado em paralelo com outros processos.
Exemplo: I'm using threads to improve the performance of my application.
Tradução: Estou usando threads para melhorar o desempenho da minha aplicação.

User interface (UI) - Interface do usuário
Explicação: A parte do aplicativo que o usuário interage, incluindo botões, menus e caixas de diálogo.
Exemplo: I'm working on improving the user interface of my application.
Tradução: Estou trabalhando em melhorar a interface do usuário da minha aplicação.
