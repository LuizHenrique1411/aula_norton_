Dicionário para iniciantes


Aqui estão algumas palavras que programadores e pessoas de áreas correlatas utilizam no seu dia a dia.

Dependency - Dependência
Explicação: Um módulo ou biblioteca que um programa depende para funcionar corretamente.
Exemplo: My program has a dependency on the requests library.
Tradução: Meu programa tem uma dependência na biblioteca requests.

Debugging - Depuração
Explicação: O processo de identificar e corrigir erros em um programa.
Exemplo: I'm spending a lot of time debugging my code today.
Tradução: Estou gastando muito tempo depurando meu código hoje.

Endpoint - Endpoint
Explicação: Uma URL específica em uma API que responde a uma solicitação HTTP.
Exemplo: I'm sending a GET request to the endpoint /users/123.
Tradução: Estou enviando uma solicitação GET para o endpoint /users/123.

Git - Git
Explicação: Um sistema de controle de versão distribuído usado para gerenciar o código-fonte de um projeto.
Exemplo: We're using Git to manage our project code.
Tradução: Estamos usando o Git para gerenciar o código do nosso projeto.

JSON - JSON
Explicação: Um formato de dados comum usado para transmitir informações pela web.
Exemplo: The API returns data in JSON format.
Tradução: A API retorna dados no formato JSON.

Library - Biblioteca
Explicação: Um conjunto de funções ou classes que podem ser reutilizadas em vários projetos.
Exemplo: I'm using the Pandas library for data analysis.
Tradução: Estou usando a biblioteca Pandas para análise de dados.

REST - REST
Explicação: Um estilo de arquitetura de software que usa solicitações HTTP para acessar e manipular recursos.
Exemplo: We're building our API using RESTful principles.
Tradução: Estamos construindo nossa API usando princípios RESTful.

Package - Pacote
Explicação: Um conjunto de módulos ou bibliotecas que fornecem funcionalidade específica.
Exemplo: I'm installing the requests package to make HTTP requests.
Tradução: Estou instalando o pacote requests para fazer solicitações HTTP.

Testing - Teste
Explicação: O processo de verificar se um programa funciona como esperado.
Exemplo: I'm writing unit tests to make sure my code is correct.
Tradução: Estou escrevendo testes unitários para garantir que meu código esteja correto.

Variable - Variável
Explicação: Um símbolo que representa um valor na memória do computador.
Exemplo: I'm storing the user's name in a variable called username.
Tradução: Estou armazenando o nome do usuário em uma variável chamada username.
