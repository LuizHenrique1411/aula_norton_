Dicionário para iniciantes


Aqui estão algumas palavras que programadores e pessoas de áreas correlatas utilizam no seu dia a dia.

ORM - Mapeamento objeto-relacional
Explicação: Um software que mapeia objetos em um programa para tabelas em um banco de dados relacional.
Exemplo: I'm using the SQLAlchemy ORM to interact with my database.
Tradução: Estou usando o ORM SQLAlchemy para interagir com meu banco de dados.

Regex - Expressões regulares
Explicação: Um padrão de caracteres usado para pesquisar e substituir texto.
Exemplo: I'm using a regex to find all occurrences of a certain word in my text.
Tradução: Estou usando expressões regulares para encontrar todas as ocorrências de uma determinada palavra no meu texto.

Scalability - Escalabilidade
Explicação: A capacidade de um sistema de lidar com um aumento na carga de trabalho.
Exemplo: We're designing our system for scalability to handle future growth.
Tradução: Estamos projetando nosso sistema para escalabilidade para lidar com o crescimento futuro.

Security - Segurança
Explicação: A proteção de um sistema contra ameaças e ataques maliciosos.
Exemplo: We're implementing strict security measures to protect our users' data.
Tradução: Estamos implementando medidas rigorosas de segurança para proteger os dados de nossos usuários.

Stack - Pilha
Explicação: Um conjunto de tecnologias usadas em conjunto para construir um aplicativo.
Exemplo: Our stack includes React for the front-end and Node.js for the back-end.
Tradução: Nossa pilha inclui React para o front-end e Node.js para o back-end.

Syntax - Sintaxe
Explicação: As regras que governam a estrutura e o formato de um código.
Exemplo: I'm correcting the syntax errors in my code.
Tradução: Estou corrigindo os erros de sintaxe no meu código.

Version control - Controle de versão
Explicação: O gerenciamento de alterações em um código-fonte ao longo do tempo.
Exemplo: We're using Git for version control of our project code.
Tradução: Estamos usando o Git para o controle de versão do código do nosso projeto.

Wrapper - Invólucro
Explicação: Uma camada adicional de código que fornece uma interface mais simples ou mais conveniente para uma funcionalidade existente.
Exemplo: I'm using a Python wrapper for the Twitter API.
Tradução: Estou usando um invólucro em Python para a API do Twitter.
