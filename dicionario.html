Dicionário para iniciantes


Aqui estão algumas palavras que programadores e pessoas de áreas correlatas utilizam no seu dia a dia.

Algorithm - Algoritmo
Explicação: Um conjunto de instruções lógicas para solucionar um problema.
Exemplo: I'm working on an algorithm to solve this math problem.
Tradução: Estou trabalhando em um algoritmo para resolver este problema de matemática.

Bug - Erro ou defeito em um programa Explicação: Um problema ou falha no programa que impede que ele funcione corretamente.
Exemplo: I found a bug in the code, so I need to fix it.
Tradução: Eu encontrei um bug no código, então preciso consertá-lo.

Code - Código
Explicação: As instruções escritas em uma linguagem de programação.
Exemplo: I need to write some code to implement this feature.
Tradução: Preciso escrever algum código para implementar esse recurso.

Debugging - Depuração
Explicação: O processo de encontrar e corrigir erros em um programa. Exemplo: I spent all day debugging this code.
Tradução: Passei o dia todo depurando esse código.

Function - Função
Explicação: Uma seção de código que realiza uma tarefa específica.
Exemplo: I wrote a function to calculate the area of a circle.
Tradução: Escrevi uma função para calcular a área de um círculo.

Loop - Loop
Explicação: Uma estrutura de controle que repete um bloco de código até que uma condição seja atendida.
Exemplo: I'm using a for loop to iterate through this array.
Tradução: Estou usando um loop for para percorrer essa matriz.

Syntax - Sintaxe
Explicação: As regras que governam a estrutura gramatical de uma linguagem de programação.
Exemplo: I need to fix the syntax errors in this code.
Tradução: Preciso corrigir os erros de sintaxe neste código.

Variable - Variável
Explicação: Um valor que pode ser armazenado e manipulado em um programa.
Exemplo: I'm using a variable to store the user's input.
Tradução: Estou usando uma variável para armazenar a entrada do usuário.

Version control - Controle de versão
Explicação: O processo de gerenciar e acompanhar as alterações feitas em um código-fonte.
Exemplo: We're using Git for version control.
Tradução: Estamos usando Git para controle de versão.

Debug log - Registro de depuração
Explicação: Uma mensagem de registro que um programa escreve para acompanhar o que está acontecendo.
Exemplo: I need to check the debug log to see what's causing this issue.
Tradução: Preciso verificar o registro de depuração para ver o que está causando este problema.
